# Nur Syahfei - Portfolio Website

A Vue 3 web portfolio showcasing my work and experience as a Flutter Developer.

## About

This is my personal portfolio website built with Vue 3 and Vite, featuring:
- Responsive design with mobile and desktop navigation
- Vertical scrolling sections
- Project showcase
- Timeline of my journey
- Privacy policy page for Ajari app

## Tech Stack

- **Framework**: Vue 3 (Composition API)
- **Build Tool**: Vite
- **Styling**: SCSS/SASS
- **Router**: Vue Router 4
- **Icons**: Custom SVG icons

## Development

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

\`\`\`bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
\`\`\`

## Features

### Home Page
- Introduction and bio

- Project cards with hover animations

- Fixed top navigation bar
- Desktop: Horizontal menu with all sections

- Smooth scroll between sections
